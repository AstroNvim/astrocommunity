# purescript Language Pack

This plugin pack does the following:

- Adds `purescript` Treesitter parsers
- Adds `purescript-language-server` language server
- Adds `purs-tidy` formatter

## LSP Configuration

The default
[purescript-language-server](https://github.com/purescript-tooling/purescript-language-server)
configuration matches all [currently supported features for Neovim
LSP](https://github.com/purescript-tooling/purescript-language-server#editor-support).

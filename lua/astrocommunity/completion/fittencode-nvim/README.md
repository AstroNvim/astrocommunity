# fittencode.nvim

Fitten Code AI Programming Assistant for Neovim

**Repository:** <https://github.com/luozhiya/fittencode.nvim>

### Default Mappings

| Mappings    | Action                 |
| ----------- | ---------------------- |
| `Tab`       | Accept all suggestions |
| `Ctrl + ⬇️` | Accept line            |
| `Ctrl + ➡️` | Accept word            |

_Note_: You need to auth first before usage - Run Fitten login <user> <password>

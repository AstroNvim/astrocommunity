# kulala.nvim

A fully-featured REST Client Interface for Neovim.

### Dependencies

- System-wide
  - curl
- Optional
  - gRPCurl (for gRPC requests)
  - Websocat (for WebSocket requests)
  - jq (to format JSON output)
  - xmllint (to format XML/HTML output)

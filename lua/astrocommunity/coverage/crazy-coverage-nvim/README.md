# Crazy Coverage

Display code coverage overlays directly in your editor with smart auto-loading and file watching.

**Repository:** <https://github.com/mr-u0b0dy/crazy-coverage.nvim>

## Features

- Multi-format support (LCOV, LLVM JSON, Cobertura XML, GCOV, LLVM Profdata)
- Smart toggle with auto-load and file watching
- Virtual text overlay with hit counts and branch coverage
- Navigation commands for covered/uncovered/partial lines
- Configurable colors and display positions

## Supported Languages

- C/C++ (GCC with gcov/lcov, LLVM/Clang)
- Any language via LCOV, Cobertura XML, or LLVM JSON formats

# Github Nvim theme

Github's Neovim themes

**Repository:** <https://github.com/projekt0n/github-nvim-theme>

## Usage

In your `plugins/astroui.lua` file, put:

```lua
return {
    "AstroNvim/astroui",
    opts = {
        colorscheme = "github_dark",
    }
}
```

Or whichever colorscheme you'd want.

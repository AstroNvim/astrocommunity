Repository: [https://github.com/brianhuster/live-preview.nvim](https://github.com/brianhuster/live-preview.nvim)

`live-preview.nvim` enables real-time preview of your `markdown`, `html`, `svg` and
`asciidoc` files directly in your web browser as you edit them in Neovim.
The plugin automatically updates the preview with every change, providing instant
feedback out-of-the-box.

> Follow this [instructions](https://github.com/brianhuster/live-preview.nvim?tab=readme-ov-file#note-for-html)
> in order to activate live preview for `html` files.

This plugin serves as a streamlined alternative to [markdown-preview.nvim](https://github.com/AstroNvim/astrocommunity/tree/main/lua/astrocommunity/markdown-and-latex/markdown-preview-nvim)
for Markdown files, with the added benefit of requiring no additional dependencies
or installation steps.
